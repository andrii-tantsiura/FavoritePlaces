# FavoritePlaces
An simple application in which you can add your favorite places with Google Maps and attach photos to them.
<br/>
Developed as part of a course on Udemy - React Native-A Practical Guide [2023]

## Developed using
- React Native
- Expo
- Redux Toolkit
- React Native Maps
- Google Maps API
- SQLite
